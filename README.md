# Revest Solutions Assignment

## Architecture

This project contains 3 applications:

1. Product Service (NestJS)
2. Order Service (NestJS)
3. Client Application (Next.js)

Order service communicates with Product service to create orders.

---

## Run Product Service

cd product-service
npm install
npm run start

Runs on http://localhost:3000

---

## Run Order Service

cd order-service
npm install
npm run start

Runs on http://localhost:3001

---

## Run Client

cd client-app
npm install
npm run dev

Runs on http://localhost:3000
